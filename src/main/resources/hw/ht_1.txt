В проект работи в классе добавлен файл DB.png, в котором описана примерная структура БД online магазина.
Создайте POJO классы для представления в Java данных каждой таблицы:
1) User - пользователи
2) Product - товары
*3) Cart - корзина
*4) CartItem - товары в корзине
5) Order - заказы
*6) OrderItem - товары в заказе
*7) Favorite - избранные товары
*8) Category - категории товаров.
Для каждой сущности создайте или переопределите обязательные методы, необходимые для нормальной работы с данными.
